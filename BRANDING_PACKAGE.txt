================================================================================
                        ANIMATIONS & TRANSITIONS
================================================================================

STANDARD TRANSITION:
  - Duration: 0.3s
  - Timing Function: ease
  - Properties: all

CUBIC BEZIER TRANSITIONS:
  - Duration: 0.3s
  - Timing Function: cubic-bezier(0.4, 0, 0.2, 1)
  - Usage: Navigation buttons, interactive elements

HOVER TRANSFORMS:
  - Cards: translateY(-4px) or translateY(-2px)
  - Links: translateX(4px)
  - Buttons: translateY(-1px) or translateY(-2px)

FADE IN UP ANIMATION:
  @keyframes fadeInUp {
    from {
      opacity: 0;
      transform: translateY(20px);
    }
    to {
      opacity: 1;
      transform: translateY(0);
    }
  }
  Duration: 0.4s
  Timing: ease-out

FADE IN SCALE ANIMATION:
  @keyframes fadeInScale {
    from {
      opacity: 0;
      transform: scale(0.95);
    }
    to {
      opacity: 1;
      transform: scale(1);
    }
  }
  Duration: 0.4s
  Timing: ease-out

STAGGERED ANIMATIONS:
  - Plan cards animate in sequence
  - Delay increments: 0.1s per item
  - Items 1-8: delays from 0.1s to 0.8s
  - Animation: fadeInUp
  - Fill Mode: both

SHIMMER/SWEEP EFFECTS:
  - Gradient sweep from left to right
  - Background: linear-gradient(90deg, transparent, rgba(211, 47, 66, 0.1-0.15), transparent)
  - Transition: left 0.4s-0.5s ease
  - Initial Position: left: -100%
  - Hover Position: left: 100%

DROPDOWN ANIMATION:
  @keyframes fadeInScale {
    from {
      opacity: 0;
      transform: scale(0.95) translateY(-8px);
    }
    to {
      opacity: 1;
      transform: scale(1) translateY(0);
    }
  }
  Duration: 0.2s
  Timing: cubic-bezier(0.4, 0, 0.2, 1)

================================================================================
                      USAGE GUIDELINES
================================================================================

WHEN APPLYING THIS BRANDING TO NEW SITES:

1. COLOR USAGE:
   - Always use rgb(211, 47, 66) as the primary accent color
   - Use gray.9 (#212121) for main backgrounds
   - Maintain white text with appropriate opacity (0.85-0.95)
   - Use red.3 (#de3e4c) for accent headings

2. TYPOGRAPHY:
   - Import Oswald from Google Fonts
   - Use font-weight 600 for headings
   - Maintain line-height ratios (1.05 for large, 1.6 for body)

3. COMPONENTS:
   - Apply backdrop-filter: blur(10px) to cards
   - Use rgba(211, 47, 66, X) for borders and accents
   - Implement consistent hover states (translateY and box-shadow)
   - Include transitions: all 0.3s ease on interactive elements

4. SPACING:
   - Use Mantine spacing variables
   - Double XL spacing for major sections
   - Consistent padding in cards and containers

5. ANIMATIONS:
   - Implement fadeInUp for content entrance
   - Use staggered delays for lists (0.1s increments)
   - Add shimmer effects to buttons and menu items
   - Keep transitions smooth with cubic-bezier easing

6. RESPONSIVE:
   - Test at sm, md breakpoints
   - Increase padding/reduce font sizes on mobile
   - Stack layouts vertically on small screens
   - Ensure touch-friendly interactive areas (min 44px)

7. CONSISTENCY:
   - Maintain border styling (1px solid rgba(211, 47, 66, 0.2))
   - Use consistent box-shadow patterns on hover
   - Apply glassmorphism to elevated surfaces
   - Keep z-index hierarchy clean
